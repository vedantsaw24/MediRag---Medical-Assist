# Medical Assist

**Medical Assist** is a smart hospital management system designed to simplify patient flow, improve doctor scheduling, and optimize hospital operations.  
It provides real-time insights into **OPD queues**, **doctor availability**, **bed occupancy**, and **ICU unit status** ‚Äî helping hospitals deliver faster, more efficient care.

---

##  Features

- **Doctor Availability Tracking:**  
  View and manage schedules for specialists like cardiac surgeons or general physicians.

- **OPD Queue Management:**  
  Smart queueing algorithm for patient appointments, reducing waiting time.

- **Bed & ICU Management:**  
  Real-time visibility into occupied and available hospital beds.

- **Hospital Interface Dashboard:**  
  Streamlined interface for staff to manage patient flow and resource allocation.

- **Scalable Design:**  
  Built to handle data efficiently and support multiple hospitals or departments.
